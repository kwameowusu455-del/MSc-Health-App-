This is a group project of 4 students
In partial fulfilment of our Masters Thesis.
We call it MSc Health App.
It is a working prototype of a database use to manage
health records to improve service delivery at
Okomfo Anokye Teaching Hospital in Ghana. 
We used NoShowAppointment dataset (: https://www.kaggle.com/datasets/joniarroba/noshowappointments) from Kaggle Website to populate the database.
